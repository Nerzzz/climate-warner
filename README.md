# Climate Warner

Climate Warner is a web application that transmits weather information for the user’s location.

### System features

It works in a simple way, the user access the web site and can verify weather information based at user’s current location. There are many features, like the climate alert, weather forecast for the next 7 days, comparison of the current day’s weather with that of the previous and following days, analysis of the weather throughout the day and more.

Click [here](climate-warner.vercel.app) to access the web app.

### Technologies

This project was developed with a focus on studying asynchronous JavaScript functions, in addition to using HTML 5 and CSS 3 in the structuring and styling of the page.
