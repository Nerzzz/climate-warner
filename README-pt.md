# Climate Warner

O Climate Warner é uma aplicação web que transmite informações climaticas da região para usuário.

### Funcionalidades do sistema

Funciona de forma simples, o usuário acessa o site e pode verificar as condições climáticas baseadas em sua localização atual.  
Existem diversas funcionalidades, como o alerta ao usuário de potenciais adversidades climáticas, previsão do tempo dos próximos 7 dias, compararação do clima do dia atual com o do dia anterior e posterior, analize do clima ao decorrer do dia e mais.

Clique [aqui](https://climatewarner.vercel.app) para acessar o site.

### Tecnologias

No desenvolvimento desse projeto, visei o estudo das funções asincrônas do JavaScript, além da utilização de HTML 5 e CSS 3 para a construção e estilização das elementos da página.

### APIs

- [Open-Meteo](https://open-meteo.com/): API climática gratuíta de código aberto.
- [geolocalizator](https://developer.mozilla.org/en-US/docs/Web/API/Geolocation_API): API de geolocalização nativa do JavaScript.
